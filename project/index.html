INTRO

PhysVis is a quick visualisation tool for converting your numeric data set(s) into physical objects with the minimum effort.

Start with the usage page for detailed information on how to use it.

We advise you to use Chrome Web Browser for the best experience. Otherwise some crucial features, such as download functionality, might not work properly on other browsers.



USAGE

PhysVis offers two different visualisation methods to choose between; Data Tower or Data Circles.

Data Tower is a 3D model generated in the file format of .stl to be printed with a 3D Printer and Data Circles is a 2D model generated in the file format of .dxf to be CNC machined or laser cut.

Getting Started

Use the "Models" section on the side-bar for accessing to the specific visualization tool.

● Use the  

CHOOSE A FILE

  button for selecting a valid CSV file to upload.

● Once you select a valid CSV file, use the  

UPLOAD

  button to visualize the data.

● You can use the  

CLEAR

  button to terminate the session and select another CSV file.

● Use the  

SAVE

  button to initiate the download process for generated physical visualization files.

CSV File Formatting

Example CSV files to test: Data Tower | Data Circles

PhysVis accepts a CSV file as a data input. The data should be in the given specific formatting to be able to be used with PhysVis. The instructions on specifications of the formatting as follows;

An example CSV file is shown below.

                       
autoData1Data2Data3Data4
2000 0.185 0.440 0.488 0.357
2001 0.385 0.314 0.308 0.317
2002 0.124 0.345 0.440 0.528
2003 0.932 0.205 0.440 0.688
2004 0.185 0.343 0.628 0.357
2005 0.305 0.100 0.488 0.437

First row of the CSV file contains the identifier and data names.

                       
autoData1Data2Data3Data4

First column contains the name of each variable.

 
2000
2001
2002
2003
2004
2005

Identifier helps specifiying the lower and upper limits of data sets. Identifier can only be 'auto', 'manual' or '%'.

For limits to be automatically determined, 'auto' option is used. In order to specify the limits manually, 'manual' option can be used. However, that requires additional rows in the CSV data.

                       
manualData1Data2Data3Data4
2000 0.185 0.440 0.488 0.357
2001 0.385 0.314 0.308 0.317
2002 0.124 0.345 0.440 0.528
2003 0.932 0.205 0.440 0.688
2004 0.185 0.343 0.628 0.357
2005 0.305 0.100 0.488 0.437
Lower 0.124 0.100 0.308 0.317
Upper 0.932 0.440 0.628 0.688

Using '%' indicates that the given data sets contain percentages and therefore lower and upper limits are automatically set to '0' and '100' respectively.

                       
%Data1Data2Data3Data4
2000 22 34 23 56
2001 90 34 38 17
2002 24 45 44 98
2003 93 20 40 68
2004 85 43 62 57
2005 35 10 4 37

A CSV file can be created with a simple text editor or various online tools can be utilized. For further information on CSV file format, refer to Wikipedia page of Comma-separated values (CSV).
